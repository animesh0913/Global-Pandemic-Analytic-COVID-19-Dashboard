# Global-Pandemic-Analytic-COVID-19-Dashboard
This project uses the Global Pandemic COVID-19 to create an interactive Power BI dashboard that addresses 20 real-world public health and policy problems. It enables stakeholders to analyze global case trends, mortality, recovery rates, and active cases, and supports decision-making through actionable insights.

**Introduction**
Governments, healthcare organizations, and researchers collect vast COVID-19 data, yet face challenges turning it into actionable insights. This project bridges the gap by visualizing the pandemic’s impact across countries and regions in an intuitive and interactive format. Users can explore demographics, temporal trends, and region-wise case distribution to understand the pandemic’s trajectory.

**Objectives**
●	Analyze pandemic trends across countries and WHO regions.
●	Identify key factors driving outbreaks and mortality.
●	Support global and local policy formulation through data insights.
●	Present clean, interactive Power BI visualizations for quick decision-making.

**Dataset Description**
●	Rows: 35,156
●	Columns: 10 (Date, Country/Region, Confirmed, Deaths, Recovered, Active, New cases, New deaths, New recovered, WHO Region)
●	Dates parsed from DD-MM-YYYY format.

**Dashboard Features**

●	KPI cards for global totals.
●	Filters for country, WHO region, and date range.
●	Time-series charts for cumulative and daily trends.
●	Interactive choropleth maps.
●	Top-10 country leaderboards.
●	Insights panel with key highlights.
●	Data export and sharing options.


**Key Insights**
●	US, Brazil, and India have highest confirmed cases.
●	Some regions with younger outbreaks show faster spread rates.

**Conclusion**
The Pandemic Pulse dashboard transforms raw pandemic data into actionable intelligence, solving 20 pressing real-world problems from outbreak detection to policy guidance. Built in Power BI, its interactive design ensures accessibility for health authorities, researchers, and the general public.
